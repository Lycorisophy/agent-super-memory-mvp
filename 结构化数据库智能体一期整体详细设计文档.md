## 1. 概述

### 1.1 背景

在企业内部，运营和运维人员经常需要查询数据库以获取业务统计、排查问题。然而：

- 数据库表结构复杂，缺少外键约束，业务逻辑全部在 Java 代码中，表间关联关系隐藏在代码和文档中。
- 现有表设计文档分散在 Markdown 和 Excel 文件中，注释不完整或过时。
- 直接编写 SQL 对非技术人员门槛较高。

一期目标：**构建一个基于自然语言的数据库查询助手**，自动理解用户问题，检索表结构知识库，生成并安全执行 SQL，返回结果。

### 1.2 设计目标

- **知识库管理**：支持上传 Markdown/Excel 表结构文档，自动解析并存入向量库和图数据库。
- **智能查询**：用户输入自然语言问题，系统检索相关表结构，生成 SQL 并可选执行。
- **安全可控**：只允许 SELECT 类只读操作，支持 dry-run 模式预览 SQL。
- **置信度标记**：区分手动上传文档（默认置信度 0.6）和数据库内省（默认 0.8），便于后续质量评估。
- **复用现有基础设施**：基于已搭建的 Milvus + Neo4j + Ollama 长期记忆系统扩展。

### 1.3 技术选型

| 组件 | 选型 | 说明 |
| :--- | :--- | :--- |
| 向量数据库 | Milvus 2.x | 存储表结构描述文本的语义向量 |
| 图数据库 | Neo4j 5.x | 存储表、字段、关联关系的结构化图谱 |
| 大语言模型 | Ollama (qwen3.5) | 生成 SQL 与自然语言交互 |
| 嵌入模型 | Ollama (qwen3-embedding:8b) | 文本向量化，维度 4096 |
| Web 框架 | FastAPI | 提供 RESTful API |
| 数据库连接 | SQLAlchemy + PyMySQL | 连接目标 MySQL 执行查询 |
| 文档解析 | pandas, openpyxl | 解析 Excel 文件；Markdown 使用正则 |

## 3. 数据模型设计

### 3.1 Milvus 集合：`schema_docs`

存储表结构描述文本的向量及元数据。

| 字段名 | 数据类型 | 说明 |
| :--- | :--- | :--- |
| `doc_id` | VARCHAR(64) | 主键，雪花ID |
| `content` | VARCHAR(4096) | 表结构描述文本（用于向量化） |
| `embedding` | FLOAT_VECTOR(4096) | 文本向量 |
| `source_file` | VARCHAR(256) | 来源文件名 |
| `table_name` | VARCHAR(64) | 关联的主表名（用于过滤） |
| `timestamp` | INT64 | 记录时间戳 |
| `confidence` | FLOAT | 置信度，上传默认0.6，内省默认0.8 |
| `source` | VARCHAR(256) | 来源类型：`upload` / `db_introspect` |
| `created_at` | INT64 | 创建时间戳 |
| `updated_at` | INT64 | 更新时间戳 |

**索引**：
- 向量索引：HNSW，COSINE
- 标量索引：`table_name`、`confidence`

### 3.2 Neo4j 图模型

#### 节点

| 标签 | 属性 | 说明 |
| :--- | :--- | :--- |
| `:Table` | `name`（表名）、`comment`、`source_doc`、`confidence`、`source_type`、`updated_at` | 表节点 |
| `:Column` | `table_name`、`name`、`type`、`nullable`、`key`、`default`、`comment` | 字段节点，复合主键 (table_name, name) |

#### 关系

| 关系名 | 方向 | 说明 |
| :--- | :--- | :--- |
| `[:HAS_COLUMN]` | Table → Column | 表包含字段 |
| `[:RELATES_TO]` | Table → Table | 表间关联关系，属性：`type`（如 `belongs_to`）、`left_col`、`right_col` |

**约束**：
```cypher
CREATE CONSTRAINT FOR (t:Table) REQUIRE t.name IS UNIQUE;
CREATE CONSTRAINT FOR (c:Column) REQUIRE (c.table_name, c.name) IS NODE KEY;
```

## 4. API 接口设计

### 4.1 录入表结构：`POST /schema/ingest`

**功能**：上传 Markdown 或 Excel 文件，或提供文本描述，解析表结构并存入知识库。

**请求参数**（`multipart/form-data`）：

| 参数 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| `file` | file | 否 | .md 或 .xlsx 文件 |
| `user_text` | string | 否 | 补充自然语言描述，最长 5000 |
| `confidence` | float | 否 | 置信度，默认 0.6 |
| `source_type` | string | 否 | 来源标记，默认 `upload` |

**响应示例**：
```json
{
  "success": true,
  "tables_imported": ["users", "orders"],
  "message": "已导入 2 张表，置信度 0.6"
}
```

### 4.2 自然语言查询：`POST /schema/query`

**功能**：输入自然语言问题，生成 SQL 并可选执行。

**请求 Body**（JSON）：
```json
{
  "query": "最近一周注册的用户有多少订单？",
  "mode": "dry_run",           // dry_run 或 auto_execute
  "min_confidence": 0.5        // 可选，过滤低置信度文档
}
```

**响应示例**（`dry_run`）：
```json
{
  "sql": "SELECT COUNT(*) FROM users u JOIN orders o ON u.id = o.user_id WHERE u.created_at > DATE_SUB(NOW(), INTERVAL 1 WEEK)",
  "explanation": "该查询统计最近一周注册用户的订单总数。"
}
```

**响应示例**（`auto_execute`）：
```json
{
  "sql": "SELECT COUNT(*) AS cnt FROM ...",
  "result": [{"cnt": 42}],
  "explanation": "..."
}
```

## 6. 安全设计

### 6.1 SQL 执行限制

- **只读账号**：连接目标 MySQL 使用仅授予 `SELECT` 权限的专用账号。
- **语句白名单**：执行前检查 SQL 是否以 `SELECT`、`SHOW`、`DESCRIBE`、`EXPLAIN` 开头。
- **禁止关键词**：拦截包含 `DROP`、`DELETE`、`INSERT`、`UPDATE`、`ALTER`、`TRUNCATE` 的语句。

### 6.2 置信度机制

- **来源标记**：上传文档 `source=upload, confidence=0.6`；未来从数据库内省获取 `source=db_introspect, confidence=0.8`。
- **查询过滤**：可在检索时设置 `min_confidence` 阈值，优先使用高可信表结构。

## 2. 表结构管理模块 `schema_manager.py`

```python
"""
表结构知识库管理模块
支持从 Markdown / Excel 导入表定义，并基于向量和图进行智能查询。
"""
from __future__ import annotations

import logging
import re
import time
from typing import Any, Dict, List, Optional

import ollama
import pandas as pd
from pymilvus import Collection, CollectionSchema, DataType, FieldSchema, utility

from config import settings
from memory_service import MemorySystem, _milvus_escape

log = logging.getLogger(__name__)

SCHEMA_COLLECTION_NAME = "schema_docs"


class SchemaManager:
    """管理表结构元数据的存储与查询。"""

    def __init__(self, memory_system: MemorySystem):
        self.memory = memory_system
        self._init_schema_collection()
        self._ensure_schema_constraints()

    def _init_schema_collection(self):
        """初始化 Milvus 中存储表结构文档的集合。"""
        if utility.has_collection(SCHEMA_COLLECTION_NAME):
            self.schema_col = Collection(SCHEMA_COLLECTION_NAME)
            self.schema_col.load()
            return

        fields = [
            FieldSchema(name="doc_id", dtype=DataType.VARCHAR, max_length=64, is_primary=True),
            FieldSchema(name="content", dtype=DataType.VARCHAR, max_length=4096),
            FieldSchema(name="embedding", dtype=DataType.FLOAT_VECTOR, dim=settings.vector_dim),
            FieldSchema(name="source_file", dtype=DataType.VARCHAR, max_length=256),
            FieldSchema(name="table_name", dtype=DataType.VARCHAR, max_length=64),
            FieldSchema(name="timestamp", dtype=DataType.INT64),
            FieldSchema(name="confidence", dtype=DataType.FLOAT),
            FieldSchema(name="source", dtype=DataType.VARCHAR, max_length=256),
            FieldSchema(name="created_at", dtype=DataType.INT64),
            FieldSchema(name="updated_at", dtype=DataType.INT64),
        ]
        schema = CollectionSchema(fields, description="Table schema documents with metadata")
        self.schema_col = Collection(SCHEMA_COLLECTION_NAME, schema)
        self.schema_col.create_index(
            "embedding",
            {"metric_type": "COSINE", "index_type": "HNSW", "params": {"M": 16, "efConstruction": 200}},
        )
        self.schema_col.create_index("table_name", {"index_type": "AUTOINDEX"})
        self.schema_col.create_index("confidence", {"index_type": "AUTOINDEX"})
        self.schema_col.load()

    def _ensure_schema_constraints(self):
        """在 Neo4j 中创建表、字段节点的约束。"""
        with self.memory._driver.session() as session:
            session.run("CREATE CONSTRAINT IF NOT EXISTS FOR (t:Table) REQUIRE t.name IS UNIQUE")
            session.run("CREATE CONSTRAINT IF NOT EXISTS FOR (c:Column) REQUIRE (c.table_name, c.name) IS NODE KEY")

    # --------------------- 文件解析 ---------------------
    def parse_markdown(self, text: str) -> List[Dict[str, Any]]:
        """从 Markdown 文本中提取表定义。"""
        tables = []
        blocks = re.split(r"\n##\s+", text)
        for block in blocks:
            if not block.strip():
                continue
            lines = block.strip().split("\n")
            header = lines[0].strip()
            table_name = header.split()[0] if header else ""
            comment_match = re.search(r"[\（\(](.*?)[\）\)]", header)
            table_comment = comment_match.group(1) if comment_match else ""

            table_lines = [l for l in lines if l.strip().startswith("|")]
            if len(table_lines) < 2:
                continue

            columns = []
            for row in table_lines[2:]:
                parts = [p.strip() for p in row.split("|")[1:-1]]
                if len(parts) >= 3:
                    col_name, col_type, col_comment = parts[0], parts[1], parts[2]
                    columns.append({
                        "name": col_name,
                        "type": col_type,
                        "nullable": True,
                        "key": "",
                        "default": None,
                        "comment": col_comment,
                    })
            if table_name:
                tables.append({
                    "name": table_name,
                    "comment": table_comment,
                    "columns": columns,
                    "relations": [],
                })
        return tables

    def parse_excel(self, file_content: bytes) -> List[Dict[str, Any]]:
        """从 Excel 二进制内容解析表结构。"""
        from io import BytesIO
        tables = []
        xl = pd.ExcelFile(BytesIO(file_content))
        for sheet_name in xl.sheet_names:
            df = pd.read_excel(xl, sheet_name=sheet_name, header=0)
            if not {"字段名", "类型", "注释"}.issubset(set(df.columns)):
                continue
            columns = []
            for _, row in df.iterrows():
                col = {
                    "name": str(row.get("字段名", "")),
                    "type": str(row.get("类型", "")),
                    "nullable": bool(row.get("是否为空", True)),
                    "key": str(row.get("键", "")),
                    "default": row.get("默认值") if pd.notna(row.get("默认值")) else None,
                    "comment": str(row.get("注释", "")),
                }
                columns.append(col)
            tables.append({
                "name": sheet_name,
                "comment": "",
                "columns": columns,
                "relations": [],
            })
        return tables

    # --------------------- 存储 ---------------------
    def ingest_tables(
        self,
        tables: List[Dict],
        source_file: str,
        user_text: str = "",
        confidence: float = 0.6,
        source_type: str = "upload",
    ) -> Dict[str, Any]:
        """将解析出的表结构存入 Milvus 和 Neo4j。"""
        if not tables:
            return {"success": False, "message": "未解析到任何表结构"}

        timestamp = int(time.time())
        milvus_rows = []

        for table in tables:
            # 构造描述文本
            lines = [f"表名：{table['name']}"]
            if table.get("comment"):
                lines.append(f"注释：{table['comment']}")
            lines.append("字段：")
            for col in table["columns"]:
                line = f"  - {col['name']} ({col['type']})"
                if col.get("comment"):
                    line += f" : {col['comment']}"
                if col.get("key"):
                    line += f" [{col['key']}]"
                lines.append(line)
            if user_text:
                lines.append(f"补充说明：{user_text}")
            doc_text = "\n".join(lines)

            # 向量化
            vec = self.memory._get_embedding(doc_text)
            doc_id = f"doc_{self.memory._id_gen.next_str()}"

            milvus_rows.append({
                "doc_id": doc_id,
                "content": doc_text[:4096],
                "embedding": vec,
                "source_file": source_file,
                "table_name": table["name"],
                "timestamp": timestamp,
                "confidence": confidence,
                "source": source_type,
                "created_at": timestamp,
                "updated_at": timestamp,
            })

            # 写入 Neo4j
            with self.memory._driver.session() as session:
                session.run(
                    """
                    MERGE (t:Table {name: $name})
                    SET t.comment = $comment,
                        t.source_doc = $source,
                        t.confidence = $confidence,
                        t.source_type = $source_type,
                        t.updated_at = $ts
                    """,
                    name=table["name"],
                    comment=table.get("comment", ""),
                    source=source_file,
                    confidence=confidence,
                    source_type=source_type,
                    ts=timestamp,
                )
                for col in table["columns"]:
                    session.run(
                        """
                        MATCH (t:Table {name: $table})
                        MERGE (c:Column {table_name: $table, name: $col_name})
                        SET c.type = $type, c.nullable = $nullable, c.key = $key,
                            c.default = $default, c.comment = $comment
                        MERGE (t)-[:HAS_COLUMN]->(c)
                        """,
                        table=table["name"],
                        col_name=col["name"],
                        type=col["type"],
                        nullable=col.get("nullable", True),
                        key=col.get("key", ""),
                        default=col.get("default"),
                        comment=col.get("comment", ""),
                    )

        # 批量插入 Milvus
        if milvus_rows:
            field_names = ["doc_id", "content", "embedding", "source_file", "table_name", "timestamp",
                           "confidence", "source", "created_at", "updated_at"]
            data = [[row[f] for f in field_names] for row in milvus_rows]
            self.schema_col.insert(data)
            self.schema_col.flush()

        return {
            "success": True,
            "tables_imported": [t["name"] for t in tables],
            "message": f"已导入 {len(tables)} 张表，置信度 {confidence}",
        }

    # --------------------- 查询 ---------------------
    def query_schema(
        self,
        user_query: str,
        mode: str = "dry_run",
        min_confidence: Optional[float] = None,
    ) -> Dict[str, Any]:
        """根据自然语言问题检索表结构，生成 SQL 并可选执行。"""
        vec = self.memory._get_embedding(user_query)

        # 构建过滤表达式
        expr = None
        if min_confidence is not None:
            expr = f"confidence >= {min_confidence}"

        hits = self.schema_col.search(
            data=[vec],
            anns_field="embedding",
            param={"metric_type": "COSINE", "params": {"ef": settings.milvus_search_ef}},
            limit=5,
            expr=expr,
            output_fields=["content", "table_name", "confidence"],
        )

        relevant_tables = set()
        schema_texts = []
        for hit in hits[0]:
            content = hit.entity.get("content")
            table = hit.entity.get("table_name")
            if content:
                schema_texts.append(content)
            if table:
                relevant_tables.add(table)

        if not relevant_tables:
            return {"error": "未找到相关表，请检查知识库是否已导入"}

        tables_info = []
        with self.memory._driver.session() as session:
            for tname in relevant_tables:
                rec = session.run(
                    """
                    MATCH (t:Table {name: $name})
                    OPTIONAL MATCH (t)-[:HAS_COLUMN]->(c:Column)
                    RETURN t.name AS name, t.comment AS comment,
                           collect({name: c.name, type: c.type, comment: c.comment, key: c.key}) AS columns
                    """,
                    name=tname,
                ).single()
                if rec:
                    tables_info.append({
                        "name": rec["name"],
                        "comment": rec["comment"],
                        "columns": rec["columns"],
                    })

        prompt = self._build_sql_prompt(user_query, tables_info)
        sql = self._call_llm_for_sql(prompt)

        if not self._is_safe_sql(sql):
            return {"error": "生成的 SQL 包含危险操作，已拦截", "sql": sql}

        result = {"sql": sql}

        if mode == "auto_execute":
            try:
                from sqlalchemy import create_engine, text
                engine = create_engine(settings.mysql_url)
                with engine.connect() as conn:
                    rows = conn.execute(text(sql)).fetchall()
                    result["result"] = [dict(row._mapping) for row in rows]
            except Exception as e:
                result["error"] = f"SQL 执行失败: {e}"

        return result

    def _build_sql_prompt(self, question: str, tables: List[Dict]) -> str:
        schema_desc = ""
        for t in tables:
            schema_desc += f"\n表 {t['name']}"
            if t['comment']:
                schema_desc += f"（{t['comment']}）"
            schema_desc += "\n字段：\n"
            for col in t['columns']:
                line = f"  {col['name']} {col['type']}"
                if col['comment']:
                    line += f"  -- {col['comment']}"
                if col.get('key') == 'PRI':
                    line += " (主键)"
                schema_desc += line + "\n"

        return f"""你是一个 MySQL 专家。根据以下表结构信息，将用户问题转换为一条 SQL 语句。
只返回 SQL 语句，不要解释。

表结构：
{schema_desc}

用户问题：{question}

SQL："""

    def _call_llm_for_sql(self, prompt: str) -> str:
        client = ollama.Client(host=settings.ollama_base_url, timeout=settings.ollama_request_timeout_s)
        resp = client.chat(
            model=settings.ollama_chat_model,
            messages=[{"role": "user", "content": prompt}],
        )
        sql = resp["message"]["content"].strip()
        sql = re.sub(r"^```sql\s*", "", sql)
        sql = re.sub(r"\s*```$", "", sql)
        return sql

    def _is_safe_sql(self, sql: str) -> bool:
        sql_upper = sql.upper().strip()
        allowed = ["SELECT", "SHOW", "DESCRIBE", "DESC", "EXPLAIN"]
        return any(sql_upper.startswith(w) for w in allowed)
```

在原有 FastAPI 应用基础上，添加 SchemaManager 初始化和新路由。

```python
# main.py 新增部分（请合并到您现有的 main.py 中）
from fastapi import FastAPI, UploadFile, File, Form, HTTPException, Request
from pydantic import BaseModel
from typing import Optional, Dict, Any
from contextlib import asynccontextmanager

from config import settings
from memory_service import MemorySystem
from schema_manager import SchemaManager

@asynccontextmanager
async def lifespan(app: FastAPI):
    # 初始化 MemorySystem
    ms = MemorySystem()
    ms.connect()
    app.state.memory = ms
    # 初始化 SchemaManager
    app.state.schema_manager = SchemaManager(ms)
    yield
    # 清理
    if app.state.memory:
        app.state.memory.close()

app = FastAPI(lifespan=lifespan)

# 原有的 /memory/conversation 路由保持不变 ...

class QueryRequest(BaseModel):
    query: str
    mode: str = "dry_run"
    min_confidence: Optional[float] = None

@app.post("/schema/ingest")
async def schema_ingest(
    request: Request,
    file: Optional[UploadFile] = File(None),
    user_text: Optional[str] = Form(None, max_length=5000),
    confidence: Optional[float] = Form(0.6),
    source_type: Optional[str] = Form("upload"),
):
    sm: SchemaManager = request.app.state.schema_manager

    if not file and not user_text:
        raise HTTPException(status_code=400, detail="至少需要上传文件或提供文本描述")

    tables = []
    source_name = "manual_input"

    if file:
        filename = file.filename
        content = await file.read()
        if filename.endswith(".md"):
            text = content.decode("utf-8")
            tables = sm.parse_markdown(text)
            source_name = filename
        elif filename.endswith((".xlsx", ".xls")):
            tables = sm.parse_excel(content)
            source_name = filename
        else:
            raise HTTPException(status_code=400, detail="仅支持 .md 或 .xlsx 文件")

    if not tables and user_text:
        # 如果只提供了文本，创建一个虚拟表存储上下文
        tables = [{"name": "_context", "comment": "", "columns": []}]

    result = sm.ingest_tables(
        tables,
        source_file=source_name,
        user_text=user_text or "",
        confidence=confidence,
        source_type=source_type,
    )
    return result

@app.post("/schema/query")
async def schema_query(request: Request, body: QueryRequest):
    sm: SchemaManager = request.app.state.schema_manager
    return sm.query_schema(
        user_query=body.query,
        mode=body.mode,
        min_confidence=body.min_confidence,
    )
